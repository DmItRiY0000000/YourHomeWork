# Моё руководство по работе с Git
* # Виды команд
1. Команда **git status** - *узнать статус происходящего* 

2. Команда **git version** - *показывает текующую версию программы* 

3. Команда **git log** - *журнал изменений , показывает версии commit* 

4. Команда **git commit** - *сохранение , фиксация статуса* 

5. Команда **git push** - *позволяет передать всё сделанное на удалённый репозиторий*

6. Команда **git chekpoint** - *используется для перехода между commit* 

7. Команда **git diff** - *позволяет увидеть разницу* 

8. Команда **git clone** - *позволяет скопировать удалённый репозиторий*

9. Команда **git pull** - *позволяет стянуть с удалённого репозитория все изменения* 

10. Команда **git add** - *для отслеживания файла*
